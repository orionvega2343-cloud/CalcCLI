# CalcCLI

Простой CLI калькулятор для терминала. Поддерживает 4 арифметические операции.

## Установка

git clone https://github.com/orionvega2343-cloud/CalcCLI.git
cd CalcCLI

## Использование

go run ./calc/main.go [флаг] число1 число2

## Примеры

# Сложение (по умолчанию)
go run ./calc/main.go 10 5
→ 10 + 5 = 15

# С указанием операции
go run ./calc/main.go -op sub 10 5
→ 10 - 5 = 5
